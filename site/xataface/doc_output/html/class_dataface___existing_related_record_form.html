Detailed Description

File: Dataface/ExistingRelatedRecordForm.php Author: Steve Hannah shann.nosp@m.ah@s.nosp@m.fu.ca Created: November 2005

Description:

A form to add existing records to a relationship. It provides a select list of existing records in the relationship domain, along with extra fields necessary to make the relationship.
