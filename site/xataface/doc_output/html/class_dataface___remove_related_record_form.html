File: Dataface/RemoveRelatedRecordForm.php Author: Steve Hannah shann.nosp@m.ah@s.nosp@m.fu.ca Created: November 2005

Description:

An HTML form to remove a set of records from a relationship. It also has an option to remove the records from the database.
